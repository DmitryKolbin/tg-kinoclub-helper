# tg-kinoclub-helper

Телеграм-бот для помощи в управлении киноклубом.

## Требования
- **Rust**: `1.80` или новее.

## Настройка окружения (ENVS)
- `TELOXIDE_TOKEN` - токен телеграм-бота
- `TMDB_API_KEY` - ключ TMDB API
- `RUST_LOG` - уровень логов (по умолчанию `info`)
- `STORE_PATH` - путь к файлу с данными (по умолчанию `movie_bot_state.json`)

## Тестирование

Для запуска тестов используйте стандартную команду cargo:

```bash
cargo test
```

Для тестов интеграции с Telegram используется `wiremock`. Чтобы избежать конфликтов при установке переменных окружения, рекомендуется запускать тесты в один поток:

```bash
cargo test -- --test-threads=1
```
